# PackVideoFix — заметки

## Что работает
- WinForms приложение показывает превью с веб-камеры (OpenCvSharp).
- Запись стартует по 1-му скану штрихкода.
- Остановка по 2-му скану того же штрихкода → видео сохраняется.
- Кнопка "СТОП" останавливает и удаляет текущую запись.
- Сканер работает глобально (IMessageFilter), фокус ставить не нужно.
- SplitContainer больше не падает при запуске (MinSize задаётся в Shown + FixSplitter).

## Где лежат файлы
- Видео: {RecordRoot}\yyyy-MM-dd\{StationName}\{barcode}_{timestamp}_{Station}.avi
- Мета: рядом JSON с полями Barcode/Station/StartedAt/FinishedAt/Status/Reason/VideoPath
- Временный файл записи: *.part.avi

## UI сейчас
- Слева превью камеры.
- Справа: заглушка Ozon-инфо + большая кнопка "СТОП".
- Сверху меню: "Настройки" (заглушка), "Поиск видео" (VideoSearchForm).
