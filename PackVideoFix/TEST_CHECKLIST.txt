# Чек-лист тестирования (MVP)

1) Приложение запускается
2) Нажали “Запустить камеру” -> появилось превью
3) Ввели TEST123 + Enter -> старт записи
4) Через ClipSeconds файл появился в папке RecordRoot\YYYY-MM-DD\StationName\
5) В списке запись со статусом OK
6) Поиск по "TEST" находит запись
7) Двойной клик открывает файл

Негативные:
- Камера отключена -> понятная ошибка
- Нет доступа к RecordRoot -> статус ERROR и сообщение
