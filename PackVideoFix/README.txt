# PackVideoFix

WinForms-приложение для видеофиксации упаковки (FBS): оператор видит живое видео, по скану/вводу номера заказа автоматически записывается клип и сохраняется на диск. Метаданные клипов сохраняются в SQLite для быстрого поиска по заказу.

## MVP (сейчас делаем)
- USB 4K веб-камера: превью в окне приложения
- Запись клипа фиксированной длительности по Enter/скану
- Сохранение: `RecordRoot\YYYY-MM-DD\StationName\`
- SQLite лог: barcode, путь, время, статус/ошибка
- Поиск по barcode и открытие файла из списка

## План далее
- IP-камера (RTSP)
- Запись в mp4/h264 при необходимости
- 2 камеры (верх + 45°), преролл

## Технологии
- WinForms на .NET (у нас .NET 10 LTS)
- OpenCvSharp (превью/захват USB камеры)
- Microsoft.Data.Sqlite (лог клипов)
