# Архитектура (MVP)

## Компоненты
- MainForm: UI (превью, ввод/скан, список клипов, поиск)
- CameraService: захват кадров с USB камеры, отдаёт кадры в UI, пишет клип
- SqliteClipStore: запись/поиск логов клипов (SQLite)
- AppConfig: настройки приложения

## Поток
Barcode Enter -> StartClip -> запись файла -> запись в SQLite -> поиск/открытие

## Запись видео
MVP: AVI + MJPG (максимально совместимо и просто).
Позже: mp4/h264 (через ffmpeg), IP RTSP.
