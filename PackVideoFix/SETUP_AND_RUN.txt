# Установка и запуск

## Нужно
- Windows 10/11
- Visual Studio (русская) с “Разработка классических приложений .NET”
- USB веб-камера

## Настройки
Локально создаётся `appsettings.json` (НЕ коммитим).
В репозитории хранится `appsettings.example.json`.

## Запуск
1) Открыть решение в Visual Studio
2) F5 (Запуск)
3) Нажать “Запустить камеру”
4) Ввести/сканировать номер -> Enter
5) Проверить запись в RecordRoot

## Если нет видео
- Поменять CameraIndex (0/1/2)
- Уменьшить Width/Height до 1920x1080
- Проверить, что камера не занята другим приложением
